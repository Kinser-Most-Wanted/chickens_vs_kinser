### Game Features:

1. **Game Name:** *Chickens vs. Kinser*

2. **Quick Restart:** Option to quickly restart the game.

3. **Multi-Model:** Multiple gameplay models to choose from.

4. **Low Animations for Performance:** Option to reduce animations for smoother gameplay on lower-end devices.

5. **Wave Speed Increase:** Waves get faster as the game progresses.

6. **Hard Storage on Browser:** Local storage required for browser play.

7. **Enemy Movement:** Enemies move from top to bottom.

8. **Money Count:** Kinser Bucks (replace PvZ coins). Featuring a dollar symbol with Kinser’s face.

9. **Currency System:** "Exceeds" currency, similar to PvZ’s Sun.

10. **Wave Progression:** Visual progress bar for wave completion.

11. **Attack Helicopters:** Replacing lawn mowers with attack helicopters for defense.

12. **Pause Button:** Ability to pause the game.

13. **Save System:** Multiple save files to choose from upon game launch.

14. **Shop System:**
   * In-round Shop: Uses *Exceeds* currency.

15. **Fast-Forward Button:** Speed up gameplay for convenience.

16. **Character Placement:** Characters are placed on a fixed tile grid (formerly plants).

17. **Character Types:** Certain character types deal damage to Kinser Robots.

18. **Starting Setup:**
   * 5 types of Kinser Robots.
   * 6 types of defense characters (additional types available in the shop).
   * 3 lanes/rows (expands to 5 in later levels).

19. **Unlockable Content:**
   * Unlockable Levels with progress tracking and storyline.
   * Unlockable Characters with new levels.

20. **Infinite Level:** Arcade-style freeplay mode featuring 3 attack helicopters for time/ease of use.

### **21. Gameplay & UX Enhancements (New)**

- **Loading Screen Animation:**
  * Turf unroll animation when loading into a level.
  * Optional fun animation: chicken crossing a road or walking across the screen.

- **Final Wave Indicator:**
  * Clear on-screen announcement when the final wave begins.

- **Tutorial Level Design:**
  * First level contains only 1 row.
  * Starts with a single pea shooter to introduce core mechanics.

- **New Character Concept:**
  * Cherry Bomb Chicken 🍒 – explosive unit for area damage.

- **Placement Cooldown System:**
  * Timer between placing the same type of plant/character.

- **Conditional Attacking:**
  * Chickens only attack when an enemy is present in their row.

- **Pre-Wave Economy Setup:**
  * Player starts with an initial currency amount.
  * Short delay before Wave 1 begins to allow resource generation.

- **SunChicken(?) Feedback Mechanic:**
  * Sun Chicken(?) visually glows when it is about to produce currency.
